Sobre HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas da web. Ao contrário de linguagens de programação (como JavaScript ou Python), HTML não é uma linguagem de lógica — ele marca o conteúdo da página para que os navegadores saibam como exibi-lo.

Estrutura básica do HTML

Todo arquivo HTML começa com uma estrutura base. Ela organiza o documento para que os navegadores entendam como renderizar o conteúdo: 


    <!DOCTYPE html>
    <html lang="pt-br">
        <head>
            <meta charset="UTF-8">
            <title>Minha Página</title>
        </head>
        <body>
            <h1>Olá, mundo!</h1>
            <p>Bem-vindo ao meu site.</p>
        </body>
    </html>

Explicando:

!DOCTYPE html: Informa que estamos usando HTML5.

html: Tag raiz de todo o HTML.

head: Contém informações da página (não visíveis ao usuário).

body: Onde o conteúdo visível é colocado.

Falando sobre tags

As tags são usadas para delimitar elementos no conteúdo da página. Elas geralmente vêm em pares: uma tag de abertura e uma de fechamento.

Exemplo:

Este é um parágrafo.

Existem centenas de tags, mas aqui vão algumas das mais comuns:

p Parágrafo

h1 a h6 Títulos (h1 é o mais importante, h6 o menos)

a Link

img Imagem

ul, ol, li Listas

div Divisão genérica

span Texto em linha

table, tr, td Tabelas

Algumas tags não precisam de fechamento, como img, br (quebra de linha) e hr (linha horizontal).

Atributos das tags

Os atributos fornecem informações adicionais sobre os elementos. Eles aparecem dentro da tag de abertura.

Exemplo com a tag de imagem:

img src="logo.png" alt="Logo da empresa" width="100"

Atributos comuns:

src Define a fonte (imagem, vídeo etc.)

href Define o link na tag a

alt Texto alternativo para imagens (acessibilidade)

class Aplica uma classe CSS

id Identificador único

style Estilização inline

target="_blank" Abre links em nova aba
